Perfcent — Business Management Platform

A full multi-tenant SaaS platform for small and medium businesses. Built from scratch: React frontend deployed on Vercel, Node.js/Express backend on Railway, PostgreSQL database via Supabase with row-level security for complete tenant isolation.

Shipped features include a complete sales pipeline (quote → invoice → payment), CRM with credit tracking, inventory manager with audit log, live chat Engagement Hub, WhatsApp broadcast marketing, and an audit trail. Each business on the platform is fully isolated from every other.

Stack: React · Vite · Node.js · Express · PostgreSQL · Supabase · Vercel · Railway

Algorithmic Systems

EA Optimizer — Algorithmic Trading System

A custom Expert Advisor for MetaTrader 5 paired with a Python optimisation framework. The system runs automated backtests to identify the optimal parameters for trading the US Tech 100, achieving a backtested +1200% return over two years on live market data.

Built with a multi-layer filter stack: H4 trend alignment, ATR percentile gate, spread filter, CLV filter, trailing stop system, and safe compounding logic. Python scripts automate the entire parameter sweep process.

Stack: MQL5 · Python · MetaTrader 5 · HTML report parsing
